AutoNexus – Project Overview

AutoNexus is a complete end-to-end used-car price prediction system built with a modern MLOps workflow. It integrates machine learning, automated pipelines, cloud infrastructure, and real-time deployment using tools like DagsHub, MLflow, Optuna, FastAPI, and AWS services. The project handles everything from data processing and model experimentation to deployment and scaling in production.

Machine Learning & MLOps

  • Scikit-learn for training regression models
  • Feature-engine for preprocessing and feature engineering
  • Optuna for hyperparameter optimization
  • MLflow for experiment tracking and model versioning
  • DagsHub for remote experiments & artifact storage

API & Deployment Layer

  • FastAPI for inference API
  • Uvicorn as the server
  • Docker for containerization
  • GitHub Actions + CodeDeploy for CI/CD
  • Optimized model loading for faster response

Cloud Infrastructure (AWS)

  • EC2 instances hosting FastAPI
  • Elastic Load Balancer for routing and health checks
  • Auto Scaling Group for scalability
  • S3 for storing recommendation artifacts
  • CloudWatch for monitoring
  • IAM roles & Parameter Store for secure secrets
